About

Hello and welcome! I graduated from Louisiana State University in 2011 with a bachelor's degree in Accounting. I have been working as a compliance auditor with the Department of the Treasury for the past 13 years. I often manage large volumes of financial data requiring data cleaning, transformation, and organization. To help make the process of data analysis more efficient, my business unit put together a team of developers to create Excel macros using Visual Basics Application (VBA). As part of this team, I developed VBA scripts for data preprocessing and hosted a class to teach peers on how to use it. I also recently completed Google's User Experience Designer certification and used the knowledge that I gained from the program to design more engaging user guides and visualizations to better improve user experiences. In my spare time, I like to create websites in order to practice my computer programming skills. Currently, I am extending my knowledge of Python to develop better data analytical tools to present to my organization for potential adoption.
